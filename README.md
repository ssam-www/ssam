# Claude Code 실전 교육 프로그램

개발을 모르는 4050 직장인, 주부, 중고등학생을 위한 Claude Code 실전 교육 프로그램.

## 프로그램 구조

```
/ (메인 랜딩)
├── curriculum/
│   ├── index.html      - 전체 커리큘럼 개요
│   ├── week0.html      - Day 0: 30분 첫 성공 세션
│   ├── week1.html      - Week 1: AI와 대화하는 법 (위임문 작성)
│   ├── week2.html      - Week 2: Claude Code 입문
│   ├── tracks.html     - Week 3~4: 가족/직장인/학생 트랙
│   ├── capstone.html   - Week 5~6: 캡스톤 (수료)
│   └── advanced.html   - 심화: CLAUDE.md, MCP, 운영
├── templates/
│   └── index.html      - 위임문 템플릿, 체크리스트, 명세서
└── recipes/
    └── index.html      - 바로 써먹는 레시피 30+개
```

## 핵심 설계 원칙

> 결과 먼저 → 이유 나중 → 터미널은 필요할 때 → MCP는 수료 후

## 수료 기준

1. 반복 업무를 위임문으로 바꿀 수 있다
2. Claude Code로 로컬 파일 자동화를 만들 수 있다
3. 결과를 검증하고 안전하게 운영할 수 있다
4. (선택) MCP로 외부 도구까지 확장할 수 있다
