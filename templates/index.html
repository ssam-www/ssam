위임문(프롬프트) 마스터 템플릿

기본 위임문 템플릿 (모든 상황에서 사용)

어떤 작업이든 이 템플릿을 채워서 Claude에게 요청하면 좋은 결과를 얻을 수 있습니다.

[목표] 내가 원하는 최종 결과: (파일/표/문서/목록 중 어떤 형식으로?) [입력] 내가 줄 데이터/파일: (어디에 있고, 어떤 형식인가?) [규칙] 분류/변환/정리 규칙: - 날짜 형식: YYYY-MM-DD - 분류 기준: (예: 식비/교통/쇼핑/기타) - 제외 조건: (예: 1,000원 이하 항목은 제외) [예외] 이런 경우는: - 형식이 다른 경우 → 별도 목록으로 - 내용을 알 수 없는 경우 → "미분류"로 표시 - 오류 발생 시 → 건너뛰고 로그에 기록 [안전] - 원본은 절대 수정하지 말고 복사본으로 처리 - 실행 전에 "어떤 파일을 어떻게 바꿀지" 계획을 먼저 보여줘 - 내가 OK 하면 그때 실행해줘 [검증] 결과 확인 방법: - 입력 개수와 출력 개수 비교해줘 - 결과 중 샘플 5개를 보여줘
사용법: 이 템플릿을 복붙한 후 < > 안의 내용을 내 상황에 맞게 수정하세요. 모든 항목을 채울 필요 없습니다. 목표, 입력, 규칙, 안전 4개만 채워도 됩니다.

======================================== 자동화 이름: [예: 주간 가족 일정 요약 자동화] 버전: v1.0 작성일: YYYY-MM-DD 작성자: 이름 ======================================== [ 목적 ] 이 자동화로 해결하는 문제: 이전에는 얼마나 걸렸나: 자동화 후 예상 시간: 사용 빈도: 매일/매주/매월 [ 입력 ] 파일 위치: inbox/ 파일 형식: (텍스트/CSV/PDF/Word 등) 파일명 패턴: (예: 카드내역_*.csv) 특이사항: [ 처리 규칙 ] 1. 2. 3. [ 출력 ] 파일 위치: output/ 파일 형식: 파일명 형식: YYYY-MM-DD_이름.확장자 [ 예외 처리 ] 빈 파일: 건너뛰고 로그에 기록 형식 오류: 별도 폴더(errors/)로 이동 알 수 없는 항목: "미분류"로 표시 [ 안전 규칙 ] 원본 백업: backup/YYYY-MM-DD/ 폴더에 복사 실행 확인: 계획 먼저 보여주고 OK 후 실행 금지 사항: 시스템 폴더 접근 금지 [ 검증 방법 ] 입출력 개수 확인: 샘플 비교: 5개 이상 특이 확인: [ 실행 방법 (3단계) ] 1. inbox/ 폴더에 처리할 파일 넣기 2. 터미널에서: claude 3. 이렇게 입력: "명세서대로 처리해줘" [ 실패 복구 ] 1. backup/ 폴더에서 원본 확인 2. output/ 폴더 결과 파일 삭제 3. inbox/ 에 원본 다시 복사 4. 재실행

CLAUDE.md 기본 템플릿 (심화용)

프로젝트 규칙 파일 (수료 후 사용)

작업 폴더에 이 파일을 만들면 Claude Code가 자동으로 읽습니다. 매번 설명하지 않아도 됩니다.

# 나의 자동화 프로젝트 규칙 # 이 파일을 작업 폴더 최상단에 CLAUDE.md 이름으로 저장하세요 ## 기본 언어 모든 설명과 결과물은 한국어로 작성한다. 오류 메시지도 한국어로 번역해서 설명한다. ## 안전 규칙 (절대 지켜야 함) 1. 원본 파일은 절대 수정하지 않는다 2. 모든 작업 전 backup/ 폴더에 복사 먼저 3. 실행 전 계획을 보여주고 확인 받는다 4. 시스템 폴더는 절대 접근하지 않는다 5. 삭제 작업은 재확인 후 진행 ## 폴더 구조 inbox/ : 처리할 입력 파일 output/ : 처리 완료 결과 backup/ : 원본 백업 (날짜 폴더로 구분) scripts/ : 재사용 자동화 스크립트 errors/ : 처리 실패 파일 ## 파일명 규칙 형식: YYYY-MM-DD_종류_설명.확장자 예시: 2024-03-15_보고서_3월영업실적.pdf 한글 파일명 대신 영문+날짜 조합 권장 ## 출력 형식 기본값 - 표는 마크다운 형식 - 날짜는 YYYY-MM-DD 형식 - 금액은 원화 표시 (예: 50,000원) - 불확실한 내용은 [확인 필요]로 표시
